# MrPython

A simplified programming environment for Python (3.x) - based on IDLE

## Installation

Git command line:

```
$ git clone https://github.com/fredokun/MrPython.git
...
$ cd MrPython
$ python3 mrpython/Application.py &
```

From zip:

```
$ unzip MrPython-master.zip
...
$ cd MrPython-master
$ python3 mrpython/Application.py &
```
